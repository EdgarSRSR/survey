# survey

REST-сервис для управления опросами.

Структура данных:

    

    Опрос

    --------------------

    Наименование опроса (nameSurvey)

    Дата начала (startDate)

    Дата окончания (endDate)

    Активность (да/нет) (activity)



    Вопрос опроса

    --------------------

    Ссылка на опрос (link)

    Текст вопроса (text)

    Порядок отображения (displayOrder)
